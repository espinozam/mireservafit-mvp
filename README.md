# MiReservaFit MVP

## Introducción

MiReservaFit es una aplicación web para la gestión digitalizada de reservas de entrenamientos personales en gimnasios.

## Tecnologías

* Backend Framework: Spring Boot
* Autenticación: JWT
* Frontend Framework: Angular 15+ (TypeScript)
* Base de datos: MySQL 8.0+
